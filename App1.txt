App1  – Workforce Time & Project Management System
 App is an enterprise‑grade workforce management solution designed to track employee work hours, shifts, and project allocations with high accuracy and performance.
Key Capabilities

Track daily work hours and shift schedules
Allocate time against multiple projects
Provide near real‑time insights into resource utilization
Secure access using Azure Active Directory authentication

Architecture & Technology Stack

React UI

Provides an intuitive and responsive user interface
Sends API requests and renders server responses dynamically


SWR Data Caching

Optimizes performance by caching API responses
Reduces redundant network calls and improves user experience


Node.js Proxy Layer

Acts as an intermediary between UI and backend services
Handles authentication, request forwarding, and response transformation


MSAL Azure AD Authentication

Ensures secure, role‑based access using Microsoft Azure Active Directory


API Server

Processes business logic
Returns validated and structured responses to the client



Impact

Improved accuracy in time tracking
Reduced latency through intelligent data caching
Enhanced security using enterprise authentication standards

🚀 AR Enrichment App – Associate Request Processing System
AR Enrichment App is a strategic enterprise application designed to enrich and process Associate Requests (AR) generated from demand planning activities conducted in the Presto planning tool.
Business Context

Customer demand is initially created in Presto, Hexaware’s planning tool
Planning considers resources, expenses, and timelines
Resource planning data is captured in Presto’s Resource Loading Sheet
Demand is converted into Associate Requests (AR) for fulfillment
AR Enrichment App enhances and processes these requests for downstream systems

Architecture & Technology Stack

React UI

Provides a clean and responsive interface
Sends AR‑related requests and displays processed responses


TanStack Query (Data Caching)

Efficiently caches server data
Improves performance and minimizes redundant API calls


Node.js Proxy Layer

Manages authentication and request routing
Acts as a secure gateway between UI and backend services


MSAL Azure AD Authentication

Ensures secure enterprise access via Azure Active Directory


Oracle API Server

Handles AR processing logic
Integrates with Oracle backend systems
Returns validated and enriched request data



Impact

Streamlined AR processing lifecycle
Reduced manual intervention in demand fulfillment
Improved system performance and security compliance
